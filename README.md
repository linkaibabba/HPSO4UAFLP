# HPSO4UAFLP
# Facility Layout Problem: Optimization Results & Visualization

This repository hosts the **benchmark results** and a **visualization tool** for the Facility Layout Problem (FLP). 

It contains the best layout solutions generated by a private meta-heuristic algorithm, utilizing two different layout representations: **FBS (Flexible Bay Structure)** and **STS (Sequence Triple String)**.
